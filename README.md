# Room Availability System

---

 ## Description
   Room Availability System est une application Java développée pour gérer la disponibilité des salles dans un environnement donné.
Ce projet sert à :

    --> gérer les salles

    --> vérifier leur disponibilité

    --> enregistrer les réservations

    --> appliquer les principes de persistence des données avec JPA / Hibernate

L’application illustre une architecture bien structurée, conforme aux bonnes pratiques de développement backend avec Java.

## Technologies utilisées
  -Java 17

  --Maven

  -JPA (Java Persistence API)

  -Hibernate

  -H2 Database

  -IntelliJ IDEA

## Conclusion
Ce système de disponibilité des salles montre une implémentation claire de la gestion des entités, des services et des réservations en utilisant JPA et Hibernate dans un contexte Java. Il constitue un bon exemple pour apprendre la persistance des données et structurer une application backend.
